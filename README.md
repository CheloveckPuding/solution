# solution
```python
import math


def main(y, z, x):
    a = (y ** 2 + z ** 3 / 56 + 61 * x) ** 6 / 69
    b = 17 * x ** 2 + math.log((y ** 3 - 54 * z - 0.01), 2) ** 3
    c = math.exp(x ** 2 - 77 * y) - z ** 8
    d = 59 * (72 * x - 68 * y ** 3 - 1) ** 2 - 27 * z ** 4
    return a / b - c / d
```
